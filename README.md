# WCF Service Project

## Proje Hakkında
Bu proje WPF Windwos Masaüstü Uygulması katmanlı mimari ile geliştirilmiştir ve MSSQL Express veritabanı ile entegre çalışmaktadır.

## Kullanılan Teknolojiler
- C#
- WPF
- MSSQL Express
- ADO.NET

## Özellikler
- CRUD işlemleri
- Service-oriented architecture
- Database bağlantısı
- SOAP servis yapısı

## Kurulum
1. SQL Server Express kurulumu
2. Database scriptini çalıştır
3. Connection string güncelle
4. Solution’u build et
5. Service host çalıştır

## Notlar
- Veritabanı dosyası repo içinde yoktur.
- Güvenlik nedeniyle connection string sanitize edilmiştir.
